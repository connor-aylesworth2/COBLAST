Data location

COBLAST+ stores everything it builds — BLAST databases, SRA downloads, search results, the database registry, and temporary scratch files — under a single data folder. Point it at a drive with plenty of free space to keep large databases off your system drive.

Active location: {{ data_dir }} (free space on this drive: {{ free_label }})

{% if pending_dir %}

A new location is saved and will be used after you restart: {{ pending_dir }}

{% endif %}
  • Takes effect on restart. Close COBLAST+ and start it again after saving.
  • No spaces in the path. BLAST+ cannot build databases under a folder whose path contains spaces (for example D:\My Data). Choose a space-free folder such as D:\COBLAST_data.
  • Existing data is not moved. Changing the location starts a fresh database registry at the new folder. Databases built at the old location stay on disk and keep working — re-add them via Databases → Register existing.
